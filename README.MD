# Custom DiscorRPC (CLI)

This app lets you make a custom Discord RPC.



## feauters
- option to include buttons
- sharable RPC's  with JSON 


# Setup 
   1. Install [NodeJS](https://nodejs.org/en/)
   2. Put your clientId in the config.json
   3. run the .bat file
